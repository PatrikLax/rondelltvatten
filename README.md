# Rondelltvätten

Detta är en redesign av en redan existerande app för rondelltvätten i Borås.
Syftet med detta projekt är att skapa en bättre och modernare app.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm start
   ```

## expo components

- Haptics
- BlurView
- Location
- Notifications

## RN components

- View
- Text
- Pressable
- Stylesheet
- Alert
- Scrollview
